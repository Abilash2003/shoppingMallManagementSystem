# 🛒 Shopping Mall Management System

A **Shopping Mall Management System** built using **Spring Boot**, **Spring Data JPA**, **MySQL**, and **REST APIs**. This project provides CRUD (Create, Read, Update, Delete) operations to manage users in a shopping mall management application.

---

## 📌 Features

- ➕ Add a new user
- 📋 View all users
- 🔍 View user by ID
- ✏️ Update user details
- ❌ Delete user
- 🌐 RESTful API architecture
- 💾 MySQL database integration
- 🔄 Spring Data JPA for database operations

---

## 🛠️ Technologies Used

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman (API Testing)

## 🧪 Testing

Use **Postman** to test all REST APIs.

Example:

- POST → Add User
- GET → Retrieve Users
- PUT → Update User
- DELETE → Delete User
